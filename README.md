## Getting Started

### Prerequisites

- Java 8 or higher
- Maven or Gradle

### Running the Application

1. Clone the repository:

   ```bash
   git clone https://github.com/Deekshanaik2004/springboot1.git
   cd springboot1
   ```

2. Build and run with Maven:

   ```bash
   mvn spring-boot:run
   ```

   Or build a JAR and run:

   ```bash
   mvn clean package
   java -jar target/*.jar
   ```
